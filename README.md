## Description
MealPrep Match is an app that helps users connect with mealprep providers. Users can choose between mealkits, mealplans and recipes; providers can add these for users to see. This app simplifies the process of finding better food options and choosing healthier eating habits.

## App Functions
1. Customer (the user with the customer role):
    1. Create/modify customer profile - Register as a customer and define  preferences, goals, and assess current eating habits.
    2. View available services - Go through plans that are filtered by your own preferences
    3. Subscribe to available services - Customers can select a provider, choose a meal plan and complete the subscription.
    4. Write reviews for subscribed services - After making a purchase, customers are able to submit a written review.
2. Provider (the user with the provider role):
    1. Create/modify provider profile - Register as a provider. add tags to show what is offered (vegan, carnivore, mealprep, recipe, etc). This allows customers to choose providers based on preferences.
    2. Create services - Recipes(steps), Mealkits(ingredients and steps) and MealPrep guides (steps for schedule) can all be published by provider. Manage pricing of content and offer a variety of free and paid options.
    3. View provider statistics -  View statistics such as saves, review rating, subscriptions, revenue and recent posts and reviews.
    4. Reply to reviews - Providers can access reviews made on their posted content, with the ability to reply to back (customers cannot reply to other reviews).
